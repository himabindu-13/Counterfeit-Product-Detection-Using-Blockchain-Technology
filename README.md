🔗 Counterfeit Product Detection Using Blockchain Technology

📌 Overview
This project is a decentralized system to detect counterfeit products using blockchain technology.
It ensures product authenticity and traceability using QR codes and smart contracts.

🚀 Features
* Product registration on blockchain
* Unique QR code generation for each product
* Secure and tamper-proof data storage
* Product verification using QR scan
* Ownership tracking

🛠️ Technologies Used
* Blockchain (Ethereum / Hyperledger Fabric)
* Smart Contracts
* Python / Node.js
* QR Code Generation
* Web Interface (HTML, CSS)

⚙️ How It Works
1. Manufacturer registers product details
2. A unique QR code is generated
3. Product data is stored on blockchain
4. Seller transfers ownership
5. Customer scans QR code to verify authenticity

📂 Project Structure
* Smart Contracts → Blockchain logic
* QR Generator → Unique product code
* Backend → Handles data processing
* Frontend → User interaction

▶️ How to Run
1. Clone the repository
2. Install dependencies
3. Deploy smart contract
4. Run backend server
5. Access frontend in browser

🎯 Objective
To prevent counterfeit products by providing a secure and transparent verification system.
